toolr example setlog
Set a log level (for demonstration only — does nothing).

Demonstrates `Literal[...]` rendering as a
`--level
{debug,info,warning}` choice in the generated `--help` output.
Usage: toolr example setlog [OPTIONS]
Options:
•  --level
  Which log level to use.


•

   Possible values: [debug, info, warning]

•


   Default: info
