Remove orphan and stale cache entries

Usage: toolr self cache prune [OPTIONS]

Options:
      --all                      Remove every cache entry
  -d, --debug                    Increase verbosity
  -q, --quiet                    Suppress non-error output
      --stale-after-days <DAYS>  Override the staleness threshold [default: 30]
      --dry-run                  Show what would be deleted without deleting
  -y, --yes                      Skip confirmation when used with --all
  -h, --help                     Print help
