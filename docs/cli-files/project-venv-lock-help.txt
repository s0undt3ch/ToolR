toolr project venv lock
Refresh tools/uv.lock without applying (wraps `uv lock`)
Usage: toolr project venv lock [OPTIONS]
Options:
• -q --quiet
  Pass --quiet to uv


• -U --upgrade
  Re-resolve every package (--upgrade)


• -P --upgrade-package <PACKAGE>
  Re-resolve a single package; pass repeatedly for multiple
